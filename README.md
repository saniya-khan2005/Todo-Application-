
# 📝 Todo App
A simple and responsive Todo application built using **React** and **Vite**. This project helps users manage their daily tasks efficiently by allowing them to add, complete, delete, and store tasks using the browser's Local Storage.

---

## 🚀 Features

* ✅ Add new tasks
* ✅ Delete tasks
* ✅ Mark tasks as completed
* ✅ Strike-through completed tasks
* ✅ Press **Enter** to add a task
* ✅ Empty input validation
* ✅ Task Counter (Total, Completed, Pending)
* ✅ Clear Completed Tasks
* ✅ Local Storage support (tasks remain after page refresh)

## ▶️ How to Run the Project

1. Clone the repository

```bash
git clone <repository-link>
```

2. Open the project folder

```bash
cd todo-app
```

3. Install dependencies

```bash
npm install
```

4. Start the development server

```bash
npm run dev
```
