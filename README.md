# 📝 React To-Do App

A simple and clean To-Do application built using **React + Vite**.  
This project demonstrates core React concepts such as state management, event handling, and component-based architecture.

---

## 🚀 Features
- Add new tasks
- Mark tasks as completed
- Delete tasks
- Clean and responsive UI

---

## 🛠 Tech Stack
- React
- Vite
- JavaScript
- CSS
